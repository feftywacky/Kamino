# Kamino
IN DEVELOPMENT <br>
Productivity Web App <br>

## Getting Started

setup environment variables
```env
CONVEX_DEPLOYMENT=

NEXT_PUBLIC_CONVEX_URL=

NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=

CLERK_SECRET_KEY=
```
<br>
run both backend and frontend
```bash
npx convex dev
npm run dev
```

open [http://localhost:3000/](http://localhost:3000/)
